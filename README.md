# quote-tool-t-c

Public-facing terms and conditions for the Kū Dynamics Quote Tool.
Hosted via GitHub Pages so the URLs can be supplied to Intuit for the
QuickBooks Online App Assessment.

## Pages

- `index.html` — landing page linking to both legal docs
- `eula.html` — end-user license agreement
- `privacy.html` — privacy policy
- `style.css` — shared minimal styling

## Hosting

Enable GitHub Pages in repo settings:

1. Repo **Settings → Pages**
2. **Source** = "Deploy from a branch"
3. **Branch** = `main`, folder = `/ (root)`
4. Save

Pages will be live at:

- `https://kudynamics.github.io/quote-tool-t-c/`
- `https://kudynamics.github.io/quote-tool-t-c/eula.html`
- `https://kudynamics.github.io/quote-tool-t-c/privacy.html`

Use the latter two URLs in Intuit's End-User License Agreement URL and
Privacy Policy URL fields during App Assessment.
